# Download/Export - Quick Start

## Untuk User Biasa

### Download Profile Saya
```
1. Login ke dashboard
2. Klik "Profile" di sidebar
3. Klik tombol "Download JSON" atau "Download TXT"
4. File otomatis terdownload
```

**Format:**
- JSON: Data terstruktur lengkap
- TXT: Format teks mudah dibaca

## Untuk Admin

### 1. Export Semua User
```
1. Login sebagai ADMIN
2. Klik "User Management" di sidebar
3. Klik "Export JSON" atau "Export CSV"
4. File otomatis terdownload
```

**Format:**
- JSON: Untuk backup/import
- CSV: Untuk Excel/Google Sheets

### 2. Download System Report
```
1. Login sebagai ADMIN
2. Buka Dashboard (halaman utama)
3. Scroll ke "System Report"
4. Klik "Download System Report"
5. File JSON terdownload
```

**Isi Report:**
- Total users, admins, regular users
- Profile completion rate
- Distribusi per prodi
- Distribusi per angkatan

## Tips

✅ File otomatis diberi nama dengan tanggal
✅ Semua download langsung, tidak perlu tunggu
✅ Data real-time dari database
✅ Aman - hanya data yang authorized

## Troubleshooting

**Tidak bisa download?**
- Pastikan sudah login
- Admin features butuh role ADMIN
- Refresh halaman jika error

**File kosong?**
- Cek apakah ada data di database
- Pastikan profile sudah diisi
