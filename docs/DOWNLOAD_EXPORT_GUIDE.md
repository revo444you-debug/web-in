# Download & Export Guide

## Overview
Web ini dilengkapi dengan fitur download/export data dalam berbagai format untuk memudahkan backup dan analisis data.

## Fitur Download

### 1. Export Profile (User)
Setiap user dapat mendownload data profil mereka sendiri.

**Lokasi:** Dashboard > Profile

**Format tersedia:**
- JSON - Data terstruktur lengkap
- TXT - Format teks yang mudah dibaca

**Data yang di-export:**
- Email
- Role
- Tanggal registrasi
- Informasi profil (nama, NIM, prodi, angkatan, nomor telepon)
- URL foto profil

**Cara menggunakan:**
1. Login ke dashboard
2. Buka halaman Profile
3. Klik tombol "Download JSON" atau "Download TXT"
4. File akan otomatis terdownload

### 2. Export Users (Admin Only)
Admin dapat mendownload data semua user dalam sistem.

**Lokasi:** Dashboard > User Management

**Format tersedia:**
- JSON - Data terstruktur untuk import/backup
- CSV - Format spreadsheet untuk Excel/Google Sheets

**Data yang di-export:**
- ID user
- Email
- Role
- Tanggal registrasi
- Data profil lengkap (nama, NIM, prodi, angkatan, nomor telepon)

**Cara menggunakan:**
1. Login sebagai ADMIN
2. Buka halaman User Management
3. Klik tombol "Export JSON" atau "Export CSV"
4. File akan otomatis terdownload

**Contoh CSV:**
```csv
ID,Email,Role,Created At,Nama,NIM,Prodi,Angkatan,Nomor Telepon
1,admin@example.com,ADMIN,2024-01-01T00:00:00.000Z,Admin User,123456,Informatika,2020,081234567890
```

### 3. System Report (Admin Only)
Admin dapat mendownload laporan statistik sistem lengkap.

**Lokasi:** Dashboard (halaman utama)

**Format:** JSON

**Data yang di-export:**
- Statistik users (total, admin, regular users)
- Statistik profiles (total, dengan foto, completion rate)
- Distribusi per prodi
- Distribusi per angkatan
- Metadata (waktu generate, generated by)

**Cara menggunakan:**
1. Login sebagai ADMIN
2. Buka Dashboard (halaman utama)
3. Scroll ke bagian "System Report"
4. Klik tombol "Download System Report"
5. File JSON akan terdownload

**Contoh Report:**
```json
{
  "generatedAt": "2024-03-05T10:30:00.000Z",
  "generatedBy": "admin@example.com",
  "statistics": {
    "users": {
      "total": 50,
      "admins": 2,
      "regularUsers": 48
    },
    "profiles": {
      "total": 45,
      "withPhoto": 30,
      "withoutPhoto": 15,
      "completionRate": "90.00%"
    }
  },
  "distributions": {
    "byProdi": [
      { "prodi": "Informatika", "count": 25 },
      { "prodi": "Sistem Informasi", "count": 20 }
    ],
    "byAngkatan": [
      { "angkatan": "2023", "count": 15 },
      { "angkatan": "2022", "count": 20 }
    ]
  }
}
```

## API Endpoints

### Export Profile
```
GET /api/export/profile?format={json|txt}
```
- Requires: Authentication
- Access: Any logged-in user

### Export Users
```
GET /api/export/users?format={json|csv}
```
- Requires: Authentication + ADMIN role
- Access: Admin only

### Export Report
```
GET /api/export/report
```
- Requires: Authentication + ADMIN role
- Access: Admin only
- Format: JSON only

## Komponen

### DownloadButton Component
Komponen reusable untuk tombol download.

**Props:**
```typescript
interface DownloadButtonProps {
  endpoint: string;        // API endpoint
  format?: 'json' | 'csv' | 'txt';  // Format file
  label?: string;          // Label tombol
  variant?: 'default' | 'outline' | 'ghost';
  size?: 'default' | 'sm' | 'lg' | 'icon';
}
```

**Contoh penggunaan:**
```tsx
<DownloadButton
  endpoint="/api/export/profile"
  format="json"
  label="Download JSON"
  variant="outline"
/>
```

## Security

### Authorization
- Profile export: Hanya user yang login
- Users export: Hanya ADMIN
- Report export: Hanya ADMIN

### Data Privacy
- User hanya bisa download data mereka sendiri
- Admin bisa download semua data (sesuai role)
- Password tidak pernah di-export
- Foto profil hanya URL, bukan file binary

## Use Cases

### Backup Data
Admin dapat secara berkala mendownload:
1. Export Users (CSV/JSON) untuk backup database
2. System Report untuk tracking pertumbuhan

### Analisis Data
- Export CSV untuk analisis di Excel/Google Sheets
- System Report untuk dashboard analytics
- Distribusi prodi/angkatan untuk perencanaan

### Compliance
- User dapat download data mereka (GDPR compliance)
- Export untuk audit trail
- Backup sebelum migrasi sistem

## Troubleshooting

### Download tidak berfungsi
- Pastikan sudah login
- Cek role (ADMIN untuk fitur admin)
- Cek console browser untuk error
- Pastikan API endpoint accessible

### File kosong atau error
- Cek koneksi database
- Pastikan ada data untuk di-export
- Cek server logs untuk error detail

### Format tidak sesuai
- Pastikan parameter format benar (json/csv/txt)
- Cek Content-Type di response header
