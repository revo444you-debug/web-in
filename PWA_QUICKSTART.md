# PWA Quick Start - Install Web Sebagai Aplikasi

## 🚀 Install dalam 30 Detik

### 💻 Di Laptop/Desktop

#### Chrome / Edge
```
1. Buka website
2. Lihat icon ➕ di address bar (kanan atas)
3. Klik icon → "Install"
4. Selesai! ✅
```

**Atau:**
```
Menu (⋮) → "Install Dashboard Template" → Install
```

#### Firefox
```
Menu (⋮) di address bar → "Install" → Install
```

#### Safari (Mac)
```
File → Add to Dock
```

---

### 📱 Di Mobile

#### Android
```
1. Buka di Chrome
2. Tap menu (⋮)
3. Tap "Install app"
4. Tap "Install"
5. Selesai! ✅
```

**Atau tunggu banner install muncul otomatis**

#### iOS
```
1. Buka di Safari
2. Tap Share (⬆️)
3. Tap "Add to Home Screen"
4. Tap "Add"
5. Selesai! ✅
```

---

## ✨ Keuntungan Install

- 📱 Akses cepat dari desktop/home screen
- 🚀 Loading lebih cepat
- 📡 Bisa digunakan offline (limited)
- 🎨 Full screen tanpa browser UI
- 💾 Data di-cache untuk performa lebih baik

---

## 🗑️ Uninstall

### Windows/Mac
```
Buka app → Menu (⋮) → "Uninstall"
```

### Android
```
Long press icon → "Uninstall"
```

### iOS
```
Long press icon → "Remove App"
```

---

## ❓ Troubleshooting

**Install button tidak muncul?**
- Gunakan Chrome/Edge/Firefox terbaru
- Reload halaman (Ctrl+F5)
- Clear cache

**Sudah install tapi tidak muncul?**
- Cek desktop/start menu/home screen
- Cari "Dashboard Template"

## 🎯 Generate Icon

1. Buka `scripts/generate-icons.html` di browser
2. Klik "Download 192x192" dan "Download 512x512"
3. Simpan di folder `public/`
4. Reload website
